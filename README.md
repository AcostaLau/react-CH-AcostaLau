Este es un proyecto desarrollado con REACT JS para el curso de coderHouse, la web es una tienda online de zapatillas (no funcional) en donde el usuario puede elegir la marca de zapatilla que mas le guste y filtrar entre ellas (nike, jordan, adidas), una vez elegida la marca el usuario podra elegir la cantidad de items que desea comprar (siempre y cuando no supere el limite de stock). Agrega al carrito y decide si seguir comprando o ir al carrito.
Una vez en el carrito se le otorgara la opcion de eliminar los items que no desee adquirir o eliminar el carrito por completo.
Una vez hecha la compra se le otorga un ID de su compra el cual esta conectado con firebase.
En firebase llega la data del comprador simulando una base de datos , que desde ahi tambien se almacenan los ITEMS (zapatillas) y hacemos un llamado a la API para poder traerlas a nuestra app.
